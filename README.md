# 48 Hours: Mapping Pakistan's Deadliest Cloudburst

An interactive scrollytelling web map documenting the August 2025 Buner cloudburst and flash floods that killed over 337 people across northern Pakistan.

## Features
- Scroll-driven narrative map (scrollytelling)
- Animated map transitions using Leaflet's flyTo()
- Dynamic scene generation from structured story data
- Scroll progress indicator
- Fully responsive design

## Tech Stack
- Leaflet.js — map rendering and animation
- Vanilla JavaScript — Intersection Observer for scroll detection
- OpenStreetMap — base map tiles

## Data Sources
NDMA, PDMA Khyber Pakhtunkhwa, Al Jazeera, Dawn, NPR, and Reuters reporting on the August 2025 Pakistan floods.
